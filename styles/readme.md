# 🎉 Festivite – Estudo de Estruturação e Estilização de Inputs

O **Festivite** é um projeto exclusivamente educacional, desenvolvido com o objetivo de praticar **estruturação de formulários em HTML** e **estilização avançada de inputs com CSS**.

> ⚠️ Este projeto não possui backend e não tem finalidade comercial.  
> O foco é apenas aprendizado de front-end.


## 📚 Objetivo do Projeto

Este projeto foi criado para estudar:

- Estruturação semântica de formulários em HTML
- Organização modular de CSS
- Customização visual de:
  - `radio`
  - `checkbox`
  - `file input`
  - `datetime-local`
  - `textarea`
- Uso de variáveis CSS (`:root`)
- Estados de validação (`:invalid`, `:focus`, `:checked`)
- Uso do seletor moderno `:has()`
- Layout com **CSS Grid** e **Flexbox**


## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- CSS Grid
- Flexbox
- CSS Custom Properties (variáveis)

---

## 🎯 O Que Está Sendo Estudado

O formulário simula a criação de um convite digital, mas serve apenas como contexto visual para praticar conceitos de estruturação e estilização.

### ✔ Estrutura de Formulário

- Uso correto de `fieldset` e `legend`
- Associação adequada entre `label` e `input`
- Organização com Grid e Flexbox

### ✔ Estilização Avançada

- Radios personalizados com animação
- Toggle switch customizado
- Checkboxes estilizados
- Upload de arquivo com botão customizado
- Feedback visual de erro
- Inputs com foco personalizado
- Controle de cores via variáveis CSS
